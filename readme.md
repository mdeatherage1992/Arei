Rails API // React Router // React Front-End // Postgres DB // Heroku Host


To Run Project = 3 running servers

NPM INSTALL (CLIENT)
BUNDLE INSTALL (API)

 First = Proxy Server for CORS COMMAND:
 cd node_modules
 node cors-anywhere/server.js

 Second = Rails Server with Postgres
 cd arei-app-api
 rails s

 Third = React Server
 cd arei-app-client
 yarn start
